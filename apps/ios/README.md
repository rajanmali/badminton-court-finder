# Smash (native iOS)

Native Swift / SwiftUI app for Smash, generated by [XcodeGen](https://github.com/yonaskolb/XcodeGen).

- **Min deployment target:** iOS 26.0
- **Language:** Swift 6.2 (default MainActor isolation, strict concurrency)
- **Bundle ID:** `com.rajanmali.smash`
- **Dependencies:** Swift Package Manager only (none yet — MapLibre/Sentry come later)

## Prerequisites

- Xcode 26 or newer (needs the iOS 26 SDK)
- XcodeGen: `brew install xcodegen`

## Setup

1. Copy the secrets template and fill in real values:

   ```bash
   cp Config/Secrets.example.xcconfig Config/Secrets.xcconfig
   ```

   Edit `Config/Secrets.xcconfig`:
   - `MAPTILER_API_KEY` — create a free key at <https://www.maptiler.com/>
   - `API_BASE_URL` — defaults to your local API (`http://localhost:3000/api/v1`)

   > These values flow xcconfig → Info.plist → `Bundle` at runtime.

2. Generate the Xcode project:

   ```bash
   xcodegen generate
   ```

3. Open it:

   ```bash
   open Smash.xcodeproj
   ```

## Build & run from the command line

```bash
xcodegen generate
xcodebuild build \
  -scheme Smash \
  -destination 'platform=iOS Simulator,name=iPhone 17' \
  CODE_SIGNING_ALLOWED=NO
```

## Run tests

Tests use [Swift Testing](https://developer.apple.com/documentation/testing).

```bash
xcodebuild test \
  -project Smash.xcodeproj \
  -scheme Smash \
  -destination 'platform=iOS Simulator,name=iPhone 17' \
  CODE_SIGNING_ALLOWED=NO
```

## Important: the generated project is NOT committed

`Smash.xcodeproj` is generated by XcodeGen and is **gitignored**. Never commit it.

Re-run `xcodegen generate` whenever you:

- pull changes that modify `project.yml`, or
- add / remove / rename source files (sources are folder-based, so new files
  are picked up automatically on the next generate — no manual project edits).

## Layout

```
apps/ios/
├── project.yml              # XcodeGen project definition (committed)
├── Config/
│   ├── Shared.xcconfig      # common settings + #include? Secrets.xcconfig
│   ├── Secrets.example.xcconfig  # template (committed)
│   └── Secrets.xcconfig     # real secrets (gitignored)
├── Smash/                   # app sources
│   ├── App/                 # @main entry + ContentView
│   ├── DesignSystem/        # design tokens (ported from packages/ui)
│   └── Resources/           # Assets.xcassets
└── SmashTests/              # Swift Testing unit tests
```
